# Jimeng Agent

即梦抖音竖屏短视频脚本生成器 — 通用 Agent Skill，根据主题生成 9:16 竖屏全维度分镜提示词 + 音效建议，可直接导入即梦 seedance 2.0 批量生成视频。

## 使用方式

将 `SKILL.md` 作为系统提示词或 Agent Skill 配置加载到任意兼容的 Agent 平台（如即梦、Reasonix、Coze、Dify 等）。

## Skill 能力

- 输入视频主题/文案 → 输出结构化大纲（含创作说明 + 钩子设计）
- 确认大纲 → 输出完整分镜提示词（7 维度画面描述 + 音效建议）
- 支持风格关键词适配（赛博科技、国潮国风、温情生活等）
- 内置负面清单：自动过滤字幕/贴纸/BGM 等不应出现在画面中的元素
